# TODO API with Nodejs

## Requirements
    * nodejs >= 10.16
    * npm >= 6.9

## Setup
    * copy .env.example to .env
    * npm install
    
## Development
    * npm run dev

## Dependencies
    * express
    * dotenv

## DevDependencies
    * nodemon
